# Plugin-StarsBarrel-1.16.5
Разработано студией @StarsDev -  TG
Инструкция по компиляции плагина для Minecraft через Maven

Установленная Java Development Kit (JDK) версии 16

Установленный Apache Maven

Установите исходный код нашего плагина

Соберите плагин через команду в терминале "mvn package"

Готовый плагин будет лежать в папке c:/users/ваше имя/IdeaProjects/StarsMessagesChat/Target

Плагин поддерживается на версии 1.16.5 paper minecraft (java) Для работы плагина понадобится: ядро сервера paper версии 1.16.5
